# JesseBakels.nl - Content Layer

This layer utilises nuxt/content writing pages in MD

## Setup

```bash
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
bun run dev
```

## Production

Build the application for production:

```bash
bun run build
```

Locally preview production build:

```bash
bun run preview
```
